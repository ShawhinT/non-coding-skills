6 ways to give
Claude context.
01
Let it search the web
Point Claude at the open internet for fresh facts and sources.
02
Have it interview you
Give a sparse prompt, ask it to interview you before answering.
03
Give it folder access
Drop files into a Project so Claude can read them on demand.
04
Plug in connectors
Let Claude pull context from Gmail, Notion, Drive, and Stripe itself.
05
Use Chrome or computer use
When no connector exists, let Claude drive the browser.
06
Build custom CLI tools
For APIs and edge cases where reliability and cost matter most.
