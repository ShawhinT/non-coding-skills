# ABA Icon System

**ABA does not use an icon font.** Every icon on the site is an **inline SVG**. Style is **Heroicons outline** at stroke `1.5`, rounded line caps/joins, `viewBox="0 0 24 24"`. A small filled subset is used for ★, ♥, the live-event dot, and social brand marks.

## Defaults

- `viewBox="0 0 24 24"`
- `fill="none"` (outline) or `fill="currentColor"` (filled)
- `stroke="currentColor"` for outline icons
- `stroke-width="1.5"`
- `stroke-linecap="round"` and `stroke-linejoin="round"`

**Sizing:**

- Inline UI: `width="20"` or `22"` (`size-5`)
- Thumbnail centerpieces (inside service-card gradient tiles): `width="48"` (`size-12`)
- Live-event dot: `10-12px` round

**Coloring:** icons inherit `currentColor` from their text container. Tint with `color: #087CA7` / `#8FC93A` / `#EE6352` / `#D19C1D` on the parent — never override `stroke=` directly.
